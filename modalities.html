Data Modalities

Understanding the different types of data in multimodal AI systems

A modality is a particular way of encoding information. In multimodal AI, we work with various data types, each with unique characteristics and processing requirements.

📝

Text

Text is sequential, discrete data representing language. It's the most common modality in AI systems.

Characteristics

  • Discrete tokens (words, subwords, characters)
  • Sequential structure with grammar
  • Rich semantic content
  • Language-dependent

Processing Techniques

Tokenization

Breaking text into tokens (BPE, WordPiece, SentencePiece)

Embeddings

Word2Vec, GloVe, contextual embeddings (BERT, GPT)

Architectures

Transformers, RNNs, LSTMs

Common Tasks

Classification Generation Translation Summarization Q&A
🖼️

Images

Images are 2D grids of pixel values capturing visual information. Computer vision has been revolutionized by deep learning.

Characteristics

  • Grid structure (height × width × channels)
  • Spatial relationships important
  • High dimensionality
  • Local patterns and global structure

Processing Techniques

Feature Extraction

CNNs, Vision Transformers (ViT), patch embeddings

Preprocessing

Normalization, augmentation, resizing

Architectures

ResNet, EfficientNet, ViT, CLIP

Common Tasks

Classification Detection Segmentation Generation Captioning
🔊

Audio

Audio is temporal waveform data. It includes speech, music, and environmental sounds with rich frequency content.

Characteristics

  • Time-series signal
  • Frequency and temporal patterns
  • Variable length
  • Multiple sources (speech, music, noise)

Processing Techniques

Feature Extraction

Spectrograms, MFCCs, mel-filterbanks

Preprocessing

Sampling, normalization, noise reduction

Architectures

Wav2Vec, Whisper, Audio Transformers

Common Tasks

Speech Recognition Speaker ID Music Analysis Sound Classification
🎬

Video

Video combines spatial (image) and temporal (sequence) information, often with accompanying audio. It's one of the most information-rich modalities.

Characteristics

  • Sequence of frames (temporal + spatial)
  • Motion and dynamics
  • Very high dimensionality
  • Often multimodal itself (visual + audio)

Processing Techniques

Feature Extraction

3D CNNs, Video Transformers, optical flow

Preprocessing

Frame sampling, temporal alignment

Architectures

TimeSformer, ViViT, SlowFast

Common Tasks

Action Recognition Video Captioning Tracking Summarization

Other Modalities

📊

Structured Data

Tables, graphs, knowledge bases with explicit relationships between entities.

🌡️

Sensor Data

Time-series from IoT devices, accelerometers, medical sensors.

🗺️

3D Data

Point clouds, meshes, depth maps for spatial understanding.

🧬

Biological Data

DNA sequences, protein structures, medical images.
